== Description ==

This plug-in support bootstrap elements. you can use that elements by shortcode.
By using this plug in, you can add almost of bootstrap elements to page and post.
So they will look more professional and good

== Installation ==

1. Upload this plug-in and set up this plug in
1. Activate the plugin through the 'Plugins' menu in WordPress
